Music Event Map
===============
Simple Google Maps and Last.fm mashup.

Build
=====
Run
```bash
$ npm install
```
then
```bash
$ gulp
```

Test
====
This library uses Mocha for tests. Mocha is hooked up to Babel via package.json's test script definition.
Run
```bash
$ npm test
```
to run Mocha with the Babel transpiler.
